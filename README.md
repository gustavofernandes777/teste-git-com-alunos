# TEMA DA AVALIAÇÃO
**Desenvolvimento em grupo de um software web**

A turma deverá desenvolver um software de gerenciamento de veterinária.
O software deverá ter:
* CRUD cliente
* CRUD animal
* CRUD veterinário
* CRUD consulta

**REGRAS**
A turma deverá seguir as seguintes regras:
* Back-end em Python, utilizando o framework Flask
* Front-end em HTML, CSS e JavaScript.
* Desenvolvimento em grupo
* Divisão de tarefas a critério do grupo
* Utilizar GitHub
* O grupo terá até o fim da aula para entregar o software
* Consultas e pesquisas na internet estão permitidas

**CRITÉRIOS DE AVALIAÇÃO**
O professor avaliará:
* Conhecimento aplicado no desenvolvimento
* Comportamento e comprometimento com a entrega do software
* Trabalho em equipe
* Resultado final do software
